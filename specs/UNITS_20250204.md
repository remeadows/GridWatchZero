## 1. Unit System Overview

### The Data Pipeline

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│   SOURCE    │ ───▶ │    LINK     │ ───▶ │    SINK     │
│  (Generate) │      │ (Transfer)  │      │  (Process)  │
└─────────────┘      └─────────────┘      └─────────────┘
      │                    │                    │
      ▼                    ▼                    ▼
   Data/tick          Bandwidth            Credits/tick
```

### Core Tension

**Source output > Link bandwidth = Packet Loss**

Players must balance upgrades across all three components to maximize throughput.

### Tier Groups

| Tier Group | Tiers | Theme | Campaign Levels |
|------------|-------|-------|-----------------|
| **RealWorld** | T1-T6 | Cybersecurity → Helix integration | 1-7 |
| **Transcendence** | T7-T10 | Post-Helix, merged with consciousness | 8-10 |
| **Dimensional** | T11-T15 | Reality-bending, multiverse access | 11-15 |
| **Cosmic** | T16-T20 | Universal scale, entropy, singularity | 16-18 |
| **Infinite** | T21-T25 | Absolute/Godlike, origin, omega | 19-20 |

### Tier Gate System

Units must reach **max level** before the next tier unlocks. Shows "MAX" badge when at tier's level cap.

### Bandwidth Formula

```
Effective Bandwidth = Base Bandwidth × Level × 1.4 × Prestige Multiplier

```


### Packet Loss Mechanics

When Source output exceeds Link bandwidth:
- **Excess data** subject to 80-100% loss
- Loss rate improves slightly with Link level
- Network defense apps can reduce packet loss during attacks

| Unit Type | Base Multiplier | Formula |

```
Final Source Output = Base × Level × 1.5 × Prestige Mult × Cert Mult

Final Credits = Processed Data × Conversion Rate × Prestige Mult × Cert Mult
```
